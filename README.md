## Overview
This project presents an SQL-based analysis of Canadian agricultural production and farm prices.  
The objective is to explore crop production, harvested area, yields, prices, and exchange rates using relational databases.

This analysis was developed as part of a professional data analytics certification and focuses on applying SQL queries to answer real-world analytical questions.

---

## Tools and Technologies
- SQL (SQLite)
- R
- RSQLite
- DBI
- Jupyter Notebook

---

## Data Description
The project uses multiple relational tables containing:
- Crop production data
- Farm price data
- Daily and monthly exchange rates (USD/CAD)

---

## Key Analyses Performed
- Creation of relational tables using SQL
- Filtering and aggregating agricultural production data
- Analysis of crop prices by type and region
- Use of joins and subqueries to combine datasets
- Interpretation of results to answer business-style questions
